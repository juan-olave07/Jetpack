# Jetpack Compose Workshops – Android Development

Repositorio con la solución de los talleres prácticos desarrollados en **Android Studio** utilizando **Kotlin** y **Jetpack Compose**, enfocados en arquitectura moderna Android, consumo de APIs, persistencia local, patrones de diseño e integración de Inteligencia Artificial.

---

## Descripción

Este repositorio contiene diferentes talleres académicos desarrollados durante el módulo de **Programación Móvil**, aplicando tecnologías modernas del ecosistema Android como:

- Jetpack Compose
- MVVM
- StateFlow
- Navigation Compose
- Retrofit
- Room Database
- Hilt
- Clean Architecture
- Material Design 3
- APIs REST
- Integración de IA
